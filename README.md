# 芋づくり製作ゲーム（SP-FARMING）
- 難しくてやることが多いゲームは、忙しい現代人にはハードルが高い！
- 何も考えずにクリックして芋を作って収穫するだけ！
- 芋を植えて、育てて、収穫するだけの **"シンプル×癒やし系×達成感"** 栽培シミュレーションゲーム！

## 概要

Flutterで開発されたシンプルな芋栽培シミュレーションゲームです。

## 機能

- **畑に植える**: 空の畑をクリックすると、芋の苗が自動的に植えられます
- **芋の状態を確認**: 成長段階（芽・若葉・成熟）を視覚的に確認できます
- **芋を収穫**: 成熟した芋をクリックして収穫できます
- **芋を保存**: 収穫した芋は自動的に倉庫に保存されます
- **倉庫内の芋を確認**: 倉庫に保存されている芋の合計個数を確認できます
- **リアルタイム成長**: プレイヤーが操作しなくても、リアル時間で芋が成長していきます

## セットアップ

### 必要な環境

- Flutter SDK (3.0.0以上)
- Dart SDK

### インストール手順

1. 依存パッケージをインストール:
```bash
flutter pub get
```

2. アプリを実行:
```bash
flutter run
```
